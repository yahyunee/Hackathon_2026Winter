## Essential Pre-Hackathon Preparation Checklist

All teams must complete the following items by **[Date]** and report to the team leader:

### ✅ Research Topic Definition
- [ ] Testable hypothesis established
- [ ] 1-page proposal completed (including objectives, model structure)

### ✅ Data Curation and Preprocessing (Most Important)
- [ ] Access permissions obtained for datasets to be used (e.g., ABCD Dataset)
- [ ] Raw Data preprocessing completed to Model-Ready format
- [ ] Backed up to external hard drive or cloud

### ✅ Environment and Role Assignment
- [ ] GPU server access and library environment tested
- [ ] Team roles assigned (core coding, data pipeline, result analysis)
- [ ] Plan B (alternative scenario) discussed
